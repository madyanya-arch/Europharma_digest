## ШАГ 1 — Создать Telegram-бота (5 минут)

1. Откройте Telegram и найдите `@BotFather`
2. Напишите ему: `/newbot`
3. Придумайте название бота, например: `Europharma Digest`
4. Придумайте username (латиницей, должен заканчиваться на `bot`), например: `europharma_digest_bot`
5. BotFather выдаст вам **токен** — строка вида `7123456789:AAHxxxxx...`
6. **Скопируйте этот токен** — он понадобится позже

### Получить свой Chat ID

1. Найдите в Telegram бота `@userinfobot`
2. Напишите ему `/start`
3. Он покажет ваш **ID** — число вида `123456789`
4. **Скопируйте этот ID**

---

## ШАГ 2 — Получить ключ Anthropic API (10 минут)

1. Перейдите на [console.anthropic.com](https://console.anthropic.com)
2. Зарегистрируйтесь или войдите
3. В левом меню: **API Keys → Create Key**
4. Назовите ключ, например: `news-bot`
5. **Скопируйте ключ** (он показывается один раз!)
6. Пополните баланс — достаточно $5 на несколько месяцев

---

## ШАГ 3 — Создать репозиторий на GitHub (10 минут)

1. Зайдите на [github.com](https://github.com) и зарегистрируйтесь
2. Нажмите **New repository** (зелёная кнопка)
3. Название: `news-bot`
4. Оставьте **Private** (приватный)
5. Нажмите **Create repository**

### Загрузить файлы

На странице нового репозитория нажмите **uploading an existing file** и загрузите три файла:

- `news_bot.py`
- `requirements.txt`

Затем создайте папки вручную через кнопку **Create new file**:

- Введите путь: `.github/workflows/digest.yml`
- Скопируйте содержимое файла `digest.yml` в редактор
- Нажмите **Commit new file**

---

## ШАГ 4 — Добавить секреты (5 минут)

В репозитории GitHub:

1. Нажмите вкладку **Settings**
2. В левом меню: **Secrets and variables → Actions**
3. Нажмите **New repository secret** и добавьте три секрета:

| Имя секрета | Значение |
|-------------|----------|
| `TELEGRAM_TOKEN` | Токен от BotFather (Шаг 1) |
| `TELEGRAM_CHAT_ID` | Ваш Chat ID (Шаг 1) |
| `ANTHROPIC_API_KEY` | Ключ Anthropic (Шаг 2) |

---

## ШАГ 5 — Проверить запуск вручную

1. В репозитории откройте вкладку **Actions**
2. Слева выберите **Утренний новостной дайджест**
3. Справа нажмите **Run workflow → Run workflow**
4. Через 1–2 минуты бот должен прислать первый дайджест в Telegram

Если что-то пошло не так — нажмите на запуск, чтобы увидеть лог ошибок.

## Настройка времени

По умолчанию бот приходит в **7:00 по Алматы**.

Чтобы изменить время, откройте файл `.github/workflows/digest.yml` и измените строку:

```
- cron: "0 2 * * *"
```

Таблица для изменения времени (UTC+5 = Алматы):

| Время (Алматы) | Cron строка |
|----------------|-------------|
| 6:00 | `0 1 * * *` |
| 7:00 | `0 2 * * *` |
| 8:00 | `0 3 * * *` |

## Частые вопросы

**Бот не пришёл — что делать?**  
Проверьте вкладку Actions в GitHub — там будет лог с причиной ошибки.

**Можно ли получать дайджест двум людям?**  
Да. Создайте Telegram-группу, добавьте туда бота и укажите ID группы в `TELEGRAM_CHAT_ID` (он будет отрицательным числом).

**Сколько стоит?**  
GitHub Actions — бесплатно (до 2000 минут в месяц, вам нужно ~2 минуты в день).  
Anthropic API — около $0.05–0.10 за один запуск, то есть **$1.5–3 в месяц**.
